# Module: context_tags

**Layer:** UI  
**Phase:** 2

## Purpose

Multi-select situational tags (work, home, social, noise, etc.).

## UI

- Chip multi-select
- Custom tag entry (calibration)
- Empty array valid

## Outputs

- `context_tags[]`

## Default tag suggestions

`work`, `home`, `social`, `alone`, `noise`, `light`, `travel`, `health`, `routine_change`

User editable in calibration_mode.
