# Module: mood_strand_visualizer

**Layer:** UI  
**Phase:** 3

## Purpose

Render crochet/blanket-style color timelines from entry color records.

## Views

- Today (Watcher home widget)
- Week band
- Optional circular layout for round display

## Input

- `(timestamp, color_hex, intensity)` tuples from local_storage

## Dependencies

- local_storage
- color-system design doc

See [color-system.md](../design/color-system.md).
